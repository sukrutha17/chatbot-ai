# 🤖 AI Chatbot

This is a simple AI chatbot built using Python and machine learning techniques.

## 🚀 Features
- Responds to user queries
- Uses TF-IDF and cosine similarity
- Simple and fast chatbot

## 🛠️ Technologies Used
- Python
- Scikit-learn

## ▶️ How to Run

1. Install dependencies:
pip install scikit-learn

2. Run:
python chatbot.py

## 💬 Example
User: hi  
Bot: Hello!

User: what is ai  
Bot: AI means Artificial Intelligence.
